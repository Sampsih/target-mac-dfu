# Политика безопасности

## Поддерживаемые версии

Исправления безопасности выпускаются для последней опубликованной версии Target Mac DFU.

## Как сообщить о проблеме

Не публикуйте сведения об уязвимости в открытом Issue. Используйте приватный отчёт GitHub Security Advisories в разделе **Security → Advisories → Report a vulnerability** репозитория.

Укажите затронутую версию, версию macOS, модель Host Mac, понятные шаги воспроизведения и ожидаемое влияние. Не прикладывайте пароли, токены и персональные данные.

## Модель доверия

Target Mac DFU запускает локальные системные инструменты, загружает каталоги и IPSW и запрашивает права администратора только для подтверждённых операций. Пароль обрабатывается системным диалогом macOS и не сохраняется приложением.
