# Участие в разработке

Спасибо за интерес к Target Mac DFU.

1. Создайте Issue и опишите проблему или изменение.
2. Сделайте fork и отдельную ветку от `main`.
3. Внесите небольшое, сфокусированное изменение.
4. Запустите `./scripts/build.sh`.
5. Проверьте интерфейс в демо-режиме.
6. Создайте Pull Request с описанием проверки и возможных рисков для DFU/Restore.

Изменения, затрагивающие Restore, повышение привилегий, загрузку исполняемых файлов или источники IPSW, должны сопровождаться объяснением модели безопасности и сценарием ручной проверки.
