# План развития

## 1.x

- автоматическая проверка обновлений приложения;
- подпись и notarization готовых сборок;
- расширенная проверка совместимости IPSW до начала Restore;
- улучшенные подсказки по кабелю и DFU-порту;
- дополнительные автоматические тесты каталогов прошивок.

## В перспективе

- управляемые корпоративные каталоги IPSW;
- экспорт отчётов по операциям;
- локализации на дополнительные языки;
- централизованная диагностика для сервисных команд.

Предложения принимаются через GitHub Issues. Порядок пунктов может меняться после тестирования на реальном оборудовании.
