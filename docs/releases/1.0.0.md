# Target Mac DFU 1.0.0

Первый публично оформленный релиз Target Mac DFU.

![Обзор Target Mac DFU](https://raw.githubusercontent.com/Sampsih/target-mac-dfu/main/docs/images/overview.png)

## Главное

- отправка подключённого Mac в DFU одной кнопкой;
- обнаружение модели и ECID;
- загрузка подписанных IPSW со скоростью, процентом, паузой и продолжением;
- поддержка стандартных, beta-, внутренних и локальных каталогов;
- Restore с двойным подтверждением полного удаления данных;
- история операций, журналы и диагностический архив;
- русский и английский интерфейс.

## Требования

- Host Mac с Apple silicon и macOS 14 или новее;
- Apple Configurator и Automation Tools (`cfgutil`);
- Xcode Command Line Tools для автоматической установки `macvdmtool`;
- прямое USB-C подключение без хаба.

## Установка

Скачайте `Target-Mac-DFU-1.0.0.zip`, распакуйте его и перенесите приложение в `/Applications`. При первом запуске нажмите по приложению с удержанием Control и выберите **Открыть**.

> **Важно:** Restore полностью стирает данные с Target Mac. Готовая сборка имеет локальную ad-hoc подпись и пока не notarized сертификатом Apple Developer.
